ESOCKS

Esocks is a command line applcation which can creating a SOCKS5 server easily. It's powered by Fucksocks

HOW TO USE

Create a SOCKS5 server at port 1080. 

java -jar esocks.jar

See usages:

java -jar esocks.jar -h
